# Installation Instructions
1. Extract zip contents
2. Click and run executable
# Operation Instructions
For Prime101, run the program, then input a number(Warning: Large values can result in the software crashing or long wait times). This number is the search range, it tells the program to search for primes between 1 and the input, eg. 100 will find all primes between 1 and 100.

For Prime, run the program, then input the prime you are looking for. It will find all primes up through that prime, eg. 100 will find all primes up through the 100th prime.
